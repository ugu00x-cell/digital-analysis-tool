## 1. 初期セットアップ

> 最初の1回だけ必要な作業です。2回目以降は不要です。

### 必要な環境
- **Windows 10 / 11** または **macOS**
- Python 3.10 以上（https://www.python.org/downloads/ からインストール）
- インターネット接続

### セットアップ手順

**① セットアップファイルをダブルクリック**

| OS | ファイル |
|----|---------|
| Windows | `初期セットアップ.bat` |
| Mac | `初期セットアップ.command` |

ダブルクリックすると必要なソフトウェアが自動的にインストールされます（数分かかります）。

> **Mac の場合：** 初回ダブルクリック時に「開発元が未確認」と表示されることがあります。
> その場合は、ファイルを右クリック →「開く」を選択してください。

**② APIキーの設定**

ツールを起動後、**設定画面**から Google Gemini APIキーを入力してください。

> APIキーは https://aistudio.google.com/app/apikey から取得できます。
> なくても基本動作しますが、フォーム解析の精度が下がります。

これでセットアップは完了です。

## 2. ツールの起動・終了

### 起動方法

**起動ファイルをダブルクリックするだけです。**

| OS | ファイル |
|----|---------|
| Windows | `起動.bat` |
| Mac | `起動.command` |

1. ターミナル画面が開きます
2. 自動的にブラウザが立ち上がり、ツール画面が表示されます
3. ブラウザが開かない場合は `http://localhost:8502` にアクセスしてください

### 終了方法

| OS | 方法 |
|----|------|
| Windows | 黒いウィンドウの × ボタンを押して閉じる |
| Mac | ターミナル画面で `Ctrl + C` を押す → ウィンドウを閉じる |

## 3. 画面の操作方法

ツールは5つのタブで構成されています。
**左から順番に操作していけば送信できます。**

### タブ① 📊 ダッシュボード

送信実績の概況が表示されます。

| 項目 | 内容 |
|------|------|
| 総送信数 | これまでの累計送信件数 |
| 成功数 | 送信が成功した件数 |
| 成功率 | 成功数 ÷ 総送信数 |
| 学習済みサイト | フォーム構造を記憶したサイト数（2回目以降の送信が高速化） |
| 今日の送信数 | 本日送信した件数 |

### タブ② 📋 リスト管理

**操作手順：**
1. 「CSVをアップロード」ボタンをクリック（またはファイルをドラッグ＆ドロップ）
2. 企業一覧がプレビュー表示されます
3. ステータスで絞り込みができます（未送信 / 送信成功 / 送信失敗 など）

### タブ③ ✉️ メッセージ設定

**送信者情報の入力：**
- 会社名、氏名、メールアドレス、電話番号を入力します
- この情報がフォームの各項目に自動入力されます

**メッセージテンプレートの編集：**
- 件名と本文を入力します
- 本文中に `{{company_name}}` と書くと、送信時に各企業名に自動置換されます

```
{{company_name}} ご担当者様

突然のご連絡失礼いたします。
株式会社●●の山田と申します。

貴社のWebサイトを拝見し、弊社のサービスが
お役に立てるのではないかと思いご連絡いたしました。
```

- 「プレビュー」を開くと、実際の送信内容を確認できます

### タブ④ 🚀 送信実行

**設定項目：**

| 設定 | 説明 |
|------|------|
| 1日の送信上限 | 1日に送信する最大件数（デフォルト: 50件） |
| ヘッドレスモード | ON: ブラウザ非表示で実行 / OFF: ブラウザ画面が見える |

> **初めて使うときは、ヘッドレスモードOFF がおすすめです。**
> ブラウザの動きを目視確認できるので、正しく動作しているか確認できます。
> 慣れてきたらONにすると、バックグラウンドで送信できます。

**送信開始：**
1. 「送信開始」ボタンを押す
2. 未送信の企業に対して順番に自動送信が始まります
3. 進捗バーと送信結果がリアルタイムで表示されます
4. 各送信の間に30〜90秒のランダムな待機時間が入ります

**送信中の表示例：**
```
🔄 株式会社ABC に送信中...
✅ 株式会社ABC: 送信完了
⏳ 45秒待機中...
🔄 サンプル商事 に送信中...
🔒 サンプル商事: CAPTCHA検出のためスキップ
```

### タブ⑤ 📄 ログ・レポート

- 全送信履歴がテーブルで表示されます
- ステータスで絞り込みができます
- 「結果CSVをダウンロード」ボタンで、送信結果をCSVファイルとして保存できます

## 4. CSVファイルの作り方

ExcelやGoogleスプレッドシートで以下の形式で作成してください。

```csv
企業名,URL
株式会社ABC,https://abc.co.jp
サンプル商事,https://sample.co.jp
テスト工業,https://test-industry.com
```

**必須カラム：**

| カラム名 | 内容 | 例 |
|---------|------|-----|
| 企業名 | 送信先の会社名 | 株式会社ABC |
| URL | 企業のWebサイトURL | https://abc.co.jp |

**注意事項：**
- URLはトップページのURLでOKです（お問い合わせページは自動で見つけます）
- URLは `https://` から始まる形式で入力してください
- Excelで作成した場合は「CSV（コンマ区切り）」形式で保存してください

## 5. 送信の流れ

ツール内部では、以下の処理が自動的に行われます。

```
1. 企業のWebサイトにアクセス
       ↓
2. 「お問い合わせ」ページを自動検出して移動
       ↓
3. CAPTCHA（ロボット防止）があるか確認
   → あった場合はスキップ（ログに記録）
       ↓
4. フォームの入力欄を自動判定
   → 会社名、氏名、メール、電話、本文の欄を識別
   → 判定できない場合はAI（Gemini）で補完
       ↓
5. フォームに自動入力して送信ボタンをクリック
       ↓
6. 送信結果（成功/失敗）をDBに記録・フォーム構造をキャッシュ
   → 次回同じドメインはキャッシュを再利用（高速化＋精度向上）
   → 失敗が増えたサイトは自動でAI再解析
       ↓
7. ランダム待機 → 次の企業へ
```

## 6. よくある質問

**Q. 送信に失敗した企業はどうなりますか？**
→ 「送信失敗」としてログに記録されます。ログ画面で確認し、必要に応じて手動で送信してください。

**Q. CAPTCHAが出た企業は？**
→ 自動スキップされ、ログに記録されます。手動での送信が必要です。

**Q. 同じ企業に2回送信されることはありますか？**
→ ありません。「送信成功」の企業は自動的にスキップされます。

**Q. Gemini APIキーがなくても使えますか？**
→ 使えます。ただし特殊なフォームの場合、精度が下がることがあります。

**Q. 1日に何件まで送信できますか？**
→ 「送信実行」画面で自由に設定できます（デフォルト: 50件）。

**Q. 送信の間隔は変更できますか？**
→ 設定画面で変更できます。設定した秒数を基準に、0.8〜1.5倍のランダム幅で自動調整されます。

**Q. どんなフォームに対応していますか？**
→ 一般的なHTMLのお問い合わせフォームに対応しています。JavaScript製の特殊なフォームは未対応の場合があります。

## 7. トラブルシューティング

### Windows

#### 「起動.bat」をダブルクリックしても何も起きない
→ Pythonがインストールされていない可能性があります。
https://www.python.org/downloads/ からPythonをインストールしてください。
インストール時に **「Add Python to PATH」にチェックを入れてください**（重要）。

#### 「ModuleNotFoundError」が表示される
→ `初期セットアップ.bat` を再度ダブルクリックしてください。

#### 「Executable doesn't exist」が表示される
→ ブラウザエンジンが未インストールです。`初期セットアップ.bat` を再度ダブルクリックしてください。

### Mac

#### 「開発元が未確認」と表示される
→ ファイルを右クリック →「開く」を選択してください。

#### 「permission denied」が表示される
→ ターミナルを開き、以下のコマンドを実行してください：
```
chmod +x 初期セットアップ.command 起動.command
```

#### 「python3: command not found」が表示される
→ Pythonがインストールされていません。
https://www.python.org/downloads/ からインストールしてください。

### 共通

#### ブラウザは開くが画面が表示されない
→ サーバーの起動に数秒かかります。10秒ほど待ってからブラウザを更新（F5）してください。

#### CSVアップロード時に「必須カラムがありません」
→ CSVの1行目（ヘッダー行）が `企業名,URL` になっているか確認してください。

#### 送信成功率が低い場合
考えられる原因：
- CAPTCHAが設置されている → 手動対応が必要です
- フォームがJavaScript製 → 一部未対応です
- URLが間違っている → CSVのURLを再確認してください

## 8. 免責事項・利用上の注意

### 法令遵守について

本ツールは、企業の公開されたお問い合わせフォームを通じて営業メッセージを送信するためのツールです。
ご利用にあたっては、以下の法令・ガイドラインを遵守してください。

- **特定電子メール法**: フォーム送信はメールとは異なりますが、相手方が送信を拒否した場合は速やかに送信を停止してください。本ツールはオプトアウト文言を自動付与しますが、停止リクエストへの対応はご利用者様の責任で行ってください。
- **不正アクセス禁止法**: 本ツールは公開されたフォームに対してのみ動作します。認証が必要なページや、利用規約で自動送信が禁止されているサイトへの使用は避けてください。
- **robots.txt**: 本ツールはrobots.txtを確認し、アクセスが拒否されているサイトには自動的に送信しません。

### 免責事項

- 本ツールの利用により発生したいかなる損害についても、開発者は責任を負いません。
- 送信先サイトの仕様変更やアクセス制限により、送信が失敗する場合があります。
- 大量送信によるIPブロックやアカウント制限のリスクはご利用者様の責任となります。
- 送信内容（営業文・送信先リスト）の適法性はご利用者様にてご判断ください。

### 複数台運用時の注意

同じネットワーク（Wi-Fi）から複数台同時に動かす場合、同一IPからのアクセスとなりブロックリスクが高まります。
複数台で運用する場合は、各端末で異なる回線（モバイル回線等）を使用することを推奨します。

### 送信成功率について

- フォーム構造はサイトごとに異なるため、100%の送信成功は保証できません。
- CAPTCHA・SPA・iframe内フォーム等、自動化が困難なサイトは自動スキップされます。
- スキップされたサイトはログに記録されますので、必要に応じて手動対応をお願いします。
- 学習機能により、使い続けることで対応サイトの成功率は向上します。
